## 📦 Установка

### Пошаговая инструкция

1. **Клонируйте репозиторий**
   ```bash
   git clone https://github.com/your-username/flower-shop-bot.git
   ```

2. **Перейдите в репозиторий**
   ```bash
   cd flower-shop-bot
   ```

3. **Установите зависимости**
   ```bash
   pip install -r requirements.txt
   ```

4. **Запустите бота**
   ```bash
   python bot/main.py
   ```
